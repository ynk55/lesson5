GIT flow

GIT clone - для того, что бы "склонить" репозиторий
GIT pull - для того, что бы получить изменения с репозитория
GIT checkout -b branchname - создать новую ветку и переключить на нее
GIT add- добавить изменения, которые сделали локально
GIT commit-m "commit text" - подписать их
GIT push origin branchname - запушить на git сервер
